# Thesis Project

Master's thesis on Austin's housing policy transformation (2020-2025).

## Directory Structure

| Directory | Purpose |
|-----------|---------|
| `Thesis_Draft/` | Active working files for the thesis |
| `Assignments_and_Proposal-SUBMITTED/` | Finalized coursework and proposal |
| `Deprecated_Writings/` | Archived content (numbered chronologically) |

## Key Files

| File | Purpose |
|------|---------|
| `TODO.md` | Prioritized task list (P1/P2/P3) |
| `GUIDELINES.md` | Project conventions and best practices |
| `CHANGELOG.md` | Session history with REVIEWED/UNREVIEWED status |

## Conventions

See `GUIDELINES.md` for full details. Key conventions:
- `-SUBMITTED`: Finalized work
- `-COMPREHENSIVE`: Long-form documents
- `-TO_INTEGRATE`: Pending integration
- `.d/`: LaTeX project directories
- `#-` prefix: Chronological order
